🔑 Cloudflare API Configuration

Connect your Cloudflare account to enable automatic DNS record updates.

Step-by-step: Generate a compatible API Token

Follow these exact steps to create a token with the right permissions

1
Log in to Cloudflare Dashboard
Go to your Cloudflare account and navigate to the API Tokens page.
🔗 dash.cloudflare.com/profile/api-tokens
2
Click "Create Token" → Use custom template
Scroll down and click Create Custom Token for fine-grained permissions.
3
Set these exact permissions
The token needs read access to list zones and edit access to update DNS records.
✓ Zone → Zone → Read
✓ Zone → DNS → Edit
4
Zone Resources: Include specific zone or All zones
For security, select only the domain(s) you'll use with this DDNS panel. Or "All zones" for convenience.
5
Create and copy the token
Cloudflare shows the token only once. Copy it immediately and paste it on the right.
# Token format looks like:
abc123xyz_AAABBBCCC-example-token-value
⚠ No token configured — DDNS inactive
🛡️ Your token is stored encrypted on the server. It is never sent to clients or exposed in the GUI after saving.

What this system will do

Using your token, the server will

🔍
Fetch your zones
List all domains on your account
✏️
Update A/AAAA records
Whenever client IP changes
🔒
Proxy status control
Orange cloud on/off per service
